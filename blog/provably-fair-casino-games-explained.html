What Does "Provably Fair" Mean?

Provably fair is a cryptographic system that allows players to independently verify that game outcomes were generated fairly and couldn't have been manipulated by the casino. It's not a marketing claim — it's mathematically verifiable proof.

In traditional online gambling, you're essentially trusting the casino's word that their random number generator (RNG) is fair. With provably fair technology, you can check the math yourself. Every game round's result is determined by a combination of cryptographic seeds that the casino commits to before you bet.

The system was pioneered by crypto casinos and has become the gold standard for transparency in online gambling. Stake Casino and BC.Game both use provably fair technology for their original games.

How Provably Fair Technology Works

The system uses a commit-reveal scheme with three cryptographic building blocks:

Step 1: The Casino Commits to a Server Seed

Before any bets are placed, the casino generates a secret "server seed" and publishes its SHA-256 hash (a fixed-length string of letters and numbers). This hash commits the casino to that specific server seed — they can't change it later because the hash would be different. You can see the hash but not the actual seed yet.

Step 2: You Provide a Client Seed

Your browser generates a "client seed" that gets combined with the server seed. This means the casino can't pre-calculate results based on their seed alone — your input is part of the equation too. Some casinos let you change your client seed for additional transparency.

Step 3: The Game Result Is Calculated

The server seed, client seed, and a "nonce" (round counter) are combined using HMAC-SHA256 to produce the game result. Each round increments the nonce, ensuring every round produces a different outcome from the same seed pair.

Step 4: You Can Verify the Result

After the round, the casino reveals the actual server seed. You can now run the same HMAC-SHA256 calculation yourself and confirm the result matches what you saw in the game. If the hash of the revealed server seed matches the hash they showed you before the round, you know they didn't cheat.

Server Seeds, Client Seeds & Nonces — Explained Simply

Think of it like this:

  • Server Seed = The casino's secret ingredient. They lock it in a "sealed envelope" (the hash) before you bet.
  • Client Seed = Your personal ingredient. You choose it, so the casino can't predict it.
  • Nonce = The round number. It ensures every game produces a different result even with the same seeds.
  • SHA-256 Hash = A digital fingerprint. It proves the casino can't change their seed after the fact.

The beauty of this system is that no single party can control the outcome. The casino can't change the server seed after your bet (because the hash is already public). And they can't predict your client seed. Both sides contribute to the randomness.

How to Verify a Game Result Yourself

Here's how to verify a Stake Casino or BC.Game result:

  1. Before playing, note the "Next Server Seed Hash" shown in the provably fair settings.
  2. Play your game round normally.
  3. After the round, the casino reveals the actual server seed and client seed used.
  4. Verify the hash — Hash the revealed server seed using SHA-256 and confirm it matches the hash shown before the round.
  5. Calculate the result — Combine the server seed, client seed, and nonce using HMAC-SHA256 to produce the game result.
  6. Compare — The calculated result should match the game result you experienced. If it does, the game was fair.

Common Myths & Misconceptions

❌ Myth: "Provably fair means no house edge"

Provably fair proves the game wasn't rigged after you bet — it doesn't remove the built-in house edge. A provably fair Crash game still has a mathematical house edge. Fairness ≠ no house advantage.

❌ Myth: "If it's provably fair, I'll win more often"

Provably fair guarantees the game wasn't altered against you, but it doesn't change the odds. You'll still lose more than you win over time (that's the house edge). The guarantee is about transparency, not profitability.

❌ Myth: "All crypto casinos are provably fair"

Not all crypto casinos use provably fair technology. Some use traditional RNGs. Always check if the casino publishes server seed hashes for its games.

✅ Truth: Provably fair is the gold standard of online gambling transparency

No other system gives you the same level of mathematical proof that a game wasn't rigged. It's a genuine innovation that puts players in control of verifying fairness.
